# 🎨 REBKA Creative Suite - Aesthetic Edition

Suite de herramientas creativas con diseño editorial aesthetic para Rebka Agency.

## 🚀 Características

- **Presentaciones Maker**: Crea PPTX profesionales con branding REBKA
- **WhatsApp Sticker Studio**: Genera stickers 512×512px con extracción de URL
- **Merchandising Designer**: Diseña productos físicos con mockups
- **Image Editor Pro**: Editor de imágenes con filtros profesionales

## 🎨 Diseño

Estilo editorial aesthetic:
- Fondo beige (#F0F0EB) con patrón de puntos
- Bordes negros gruesos (2px)
- Tipografía serif (Playfair Display) para títulos
- Accent color rosa REBKA (#FF6B9D)
- Sombras duras tipo papel

## 🛠️ Tecnologías

- **Frontend**: React + TypeScript + Vite + Tailwind CSS
- **Backend**: Python Flask
- **Estilo**: Editorial aesthetic adaptado de Google AI Studio

## 📦 Instalación

```bash
# Frontend
cd rebka-suite-aesthetic
npm install
npm run dev

# Backend
pip install -r server/requirements.txt
python server/app.py
```

## 🔗 URLs

- Frontend: http://localhost:3000
- Backend: http://localhost:5000
